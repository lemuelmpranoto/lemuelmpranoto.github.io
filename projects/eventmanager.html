Event Manager App

An event networking webapp hosted on Amazon Web Services.

TECHNOLOGIES USED

HTML5 and CSS for front-end web build.
JavaScript for application functionality, dynamic rendering and script injection.
React for front-end UI development.
Bootstrap for responsive front-end enhancements.
Amazon S3 for storing static files.
DynamoDB as database storage.
NodeJS Lambda function for database manipulation.
RESTful APIs for data communication between front and back-end.
Axios for HTTP Requests.
Amazon Cognito User Pool for user management, including OTP verification.
Git and GitHub for version control.

FEATURES

Signup, login, logout to your account.
OTP verification on account creation.
Home page as app landing page.
News panel to view latest news around the community.
Events page to view upcoming events.
Enroll/attend to upcoming events.
Post new events planned.
