CHAT'R

A social messaging webapp built with NodeJS.

TECHNOLOGIES USED

HTML5 and CSS for front-end web build.
JavaScript for application functionality, dynamic rendering and script injection.
NodeJS for handling server side and host client-side application.
MongoDB as database storage.
Git and GitHub for version control.

FEATURES

Signup, login, logout to your account.
Edit your profile including display image, name, username, bio and password.
Contact list to select who you want to chat with.
Chatroom with the ability to upload files.
OTHER PROJECTS
  • Chatr
  • Event Manager
  • Line-Following Robot

  • DE-Booth
  • Home Security System
  • Personal Website V1
